Margin & Padding

We also use a set of margin and padding classes to help with quick overwrites and positioning. 


  // Margin
  .margin-left-[5-25] { margin-left: [5-25]px !important; }
  .margin-right-[5-25] { margin-left: [5-25]px !important; }
  .margin-top-[5-25] { margin-left: [5-25]px !important; }
  .margin-bottom-[5-25] { margin-left: [5-25]px !important; }
  .noMargin { margin: 0px !important; }

  // Padding
  .padding-left-[5-25] { margin-left: [5-25]px !important; }
  .padding-right-[5-25] { margin-left: [5-25]px !important; }
  .padding-top-[5-25] { margin-left: [5-25]px !important; }
  .padding-bottom-[5-25] { margin-left: [5-25]px !important; }
  .noPadding { padding: 0px !important };

Display Classes

Quick classes to change element display. 


  .ig-inline-block { display: inline-block; }
  .ig-inline       { display: inline; }
  .ig-block        { display: block; }
  .ig-flex         { display: flex; }

Position Classes

Quick classes to change element display. 


  .ig-relative  { position: relative; }
  .ig-static    { position: static; }
  .ig-fixed     { position: fixed; }

Typography Classes

Quick classes to change font styling. 


  .ig-bold     { font-weight: bold; }
  .ig-regular  { font-weight: normal; }
  .ig-italic   { font-style: italic; }

  .ig-ell {
      white-space: nowrap;
      overflow: hidden;
      text-overflow: ellipsis;
  }

  .ig-break-word { word-wrap: break-word; }
  .ig-no-wrap    { white-space: nowrap; }

  .ig-text-light { color: $medium-gray; font-size: .8em; }

  .ig-full { width:100% };
